## what's inside

- **Next.js 16** (App Router) + **React 19** + **TypeScript** — because life's too short for pages router
- **Tailwind CSS v4** — no config file, just vibes and tokens
- **Framer Motion** — everything moves, nothing jitters
- **Nodemailer** — contact form that actually sends emails (wild concept)
- Brutalist design system: acid yellow `#DFE104`, zero border radius, zero mercy

---

## running it locally

```bash
npm install
npm run dev
```

Hit [http://localhost:3000](http://localhost:3000). You'll know it worked.

For the contact form to send emails, drop a `.env.local` at the root:

```
SMTP_HOST=...
SMTP_PORT=...
SMTP_USER=...
SMTP_PASS=...
SMTP_TO=...
```

---

## editing content

All copy lives in `lib/content.ts` — one file, every string on the site.  
Don't want to touch code? Go to `/admin` for a UI that lets you edit and save everything live.
